# AI Agent Instructions

This repository is a static website built with Astro 5, React 19, and Tailwind CSS v4.

## Global Rules

- Use Yarn 4.x via Corepack; do not use npm or pnpm.
- Build with `yarn build`.
- There is no dedicated `typecheck` script in `package.json`.
