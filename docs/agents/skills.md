# Agent Skill Requirements

Use these skills explicitly when the task matches:

- Apply `astro-framework` when working with Astro routing, layouts, content collections, islands, or rendering strategy.
- Apply `astro-framework` for i18n tasks touching Astro routing, static locale generation, content collections, or page/layout locale handling.
- Apply `frontend-design` when creating or editing UI components or pages.
- Apply `vercel-react-best-practices` when creating, editing, reviewing, or refactoring React UI.
- Apply `web-design-guidelines` when auditing or reviewing UI, and as the final audit step for UI creation or editing work.

For installed skill descriptions, see [docs/AGENT_SKILLS.md](../AGENT_SKILLS.md).
